🌐 複数端末で共有（Google Sheets 連携）
Google Apps Script (GAS) で公開した Web App URL を登録すると、入居者・体重記録・検知ルールが
クラウド（Google Sheets）経由で他の端末と自動同期されます。
GASをデプロイして取得したURLを入力して「URLを保存・接続」を押してください。 

    

      うまく繋がらない時のチェック項目
      
    
        
  1. 「アクセスできるユーザー」が「全員」になっているか
    
            Apps Script → デプロイの管理 → 編集（鉛筆）→ 「アクセスできるユーザー = 全員」 → 更新
    2. 
        
  2. URL は /exec で終わっているか
    
            /dev（テスト用）はオーナー本人しか叩けません
    3. 
        
  3. 初回承認が完了しているか
    
            Apps Script で一度「実行」を押し、「アクセスを承認」→ アカウント選択 →
            「詳細」→「（プロジェクト名）に移動」→「許可」までやり切ったか確認
    4. 
        
  4. doGet が ContentService 経由で JSON を返しているか
    
            ContentService.createTextOutput(JSON.stringify(...)).setMimeType(ContentService.MimeType.JSON)
    5. 
        
  5. うまくいかない時は「🩺 接続診断」を押して具体的な原因を表示
    6.
